Windows Subsystem for Linux

  • WSL provides a Linux-compatible kernel interface developed by Microsoft (containing no Linux kernel code), with user-mode binaries from Ubuntu running on top of it – a Bash shell and command language, with native Linux command-line tools (sed, awk, etc.) and programming language interpreters (Ruby, Python, etc.).
  • available on 64-bit Windows 10
  • Microsoft envisages WSL as "primarily a tool for developers – especially web developers and those who work on or with open source projects."
